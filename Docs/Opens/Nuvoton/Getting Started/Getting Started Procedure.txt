3. Starting to Use NuTiny-SDK-NUC200 on the Keil µVision IDE

[X] 3.1 Downloading and Installing Keil µVision IDE Software
	Please connect to the Keil company website (http://www.keil.com) to download the Keil µVision IDE and install the RVMDK.

3.2 Downloading and Installing Nuvoton Nu-Link Driver
	Please connect to Nuvoton NuMicroô website (http://www.nuvoton.com/NuMicro) to download the ìNuMicroô Keil µVisionÆ IDE driverî file. Please refer to section 6.1 for the detailed download flow. After the Nu-Link driver is downloaded, please unzip the file and execute the ìNu-Link_Keil_Driver.exeî to install the driver.
