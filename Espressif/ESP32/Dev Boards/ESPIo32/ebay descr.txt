Description:
ESPio32 is a compact wifi and bluetooth development board based on the ESP32 wifi bluetooth dual core SoC from Espressif. It features a CP2102 USB UART bridge, auto reset and programming circuit, reset and programming side button and a white user LED (IO5). RTC crystal has been added so no external RTC module is required, VP VN capacitors are implemented so IO34 and IO35 are ready for capacitive touch usage. Onboard LDO 3.3V voltage regulator, ESD protection diode.

Main chip:Espressif ESP32D0WQ dual core 240MHz
USB UART : CP2102
Flash size : 4MByte (W25Q32FVSS) 
Side buttons for reset and prog
RTC crystal included, no need for external RTC module
Solder jumper selectable flash power supply source (3.3V-default or IC).
Onboard MIC5219 3.3V LDO regulator
Onboard auto-reset and programming circuit
Fully compatible with Arduino IDE
