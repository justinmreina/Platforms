@date 		10/18/20

@section 	Notes
	1. Download and install nRF Connect for Desktop. 
	
	2.  Insert the nRF52840 Dongle in a USB port on your computer. 
			(press reset?)

		The status light (LD2) starts pulsing red, 
			Indicating that the Dongle is powered up and is in bootloader mode. 
		After a few seconds, the computer will recognize the Dongle as a USB composite device. 
			The driver needed for the nRF52840 USB DFU feature is also installed.

	3. Download the nRF5 SDK. (Optional)
		Contains application examples for the Dongle.
		For documentation, see nRF5 SDK v15.3.0.
